¿Qué es A-MATE UWU?

A-MATE UWU es una página creada para apoyar a aquellos estudiantes de nivel primaria en la materia de matemáticas, buscamos hacer mas ameno el aprendizaje de los diferentes temas que sabemos podrían llegar a ser confusos para los mas pequeños, llegando a ser de esta manera una página intuitiva, fácil de entender, con temas bien explicados, ejercicios y videos para resolver las diferentes dudas las cueles pueden llegar a surgir.
